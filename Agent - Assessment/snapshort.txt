WHICH EXECUTION CONTEXT created the checkpoint, not WHAT CAUSED IT TO PAUSE.
'loop'	    : Normal graph execution    - After executing a node in the main loop
'input'	    : User input	            - When you call invoke() with new state
'update'	: Manual state update	    - When you manually update the state
'interrupt'	: Interrupt triggered	    - When graph pauses due to interrupt
